# Elevator(Uaban Ace) LCD Emulator

日立製の乗用エレベーター、アーバンエース(第6世代,第7世代)の液晶画面をWPFを用いて再現するプロジェクトです

## <span style="color: red;">重要 Twitter APIについて</span>

Twitter APIが今後有料化されるため、場合によってはAPI連携機能を廃止する可能性があります
